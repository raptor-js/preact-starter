## Raptor Framework

See more information about the Raptor framework here: <a href="https://raptorframework.com">https://raptorframework.com</a>.

## Introduction

The following project contains a basic starter kit using [Vite](https://vite.dev) for [Raptor](https://raptorframewwork.com) and [Preact](https://preactjs.com). The project is deliberately simple as to give total freedom to the developer.
